# KICKPAD

<img width="850" height="598" alt="Screenshot 2026-05-09 040714" src="https://github.com/user-attachments/assets/b592b484-cd56-4505-8532-89b6d1ce26a7" />

This is a Xiao Rp2040  Based MicroPad , It has 9 Switches , a roter encode and a oled screen . We used matrix for switches so it uses less gpio pins and it has 4 layers the rotor key is used to change the layers which is displaced on the screen so in total we have  keys output 9x4=36 + 4 outputs from rotor = 40 Outputs.
## Features

- Based on Xiao Rp2040
- Has total 40 Outputs
## Firmware
- Firmware is based on QMK.
